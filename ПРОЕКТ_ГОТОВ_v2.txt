🎮 НОВЫЙ РАБОЧИЙ ПРОЦЕСС (v2.0)
═══════════════════════════════════════════════════════════

Шаг 1: Загрузите карту
  📁 Загрузить карту → выберите изображение Kenshi

Шаг 2: Создайте города
  🏙️ Добавить город → кликните на карте → введите название
  (Повторите для всех нужных городов)

Шаг 3: Создайте маршруты
  ПКМ по городу → кликайте по другим городам → Enter

Шаг 4: Добавьте товары
  Клик по линии маршрута → + Добавить товар
  Введите название и наценку %

Шаг 5: Сохраните
  💾 Сохранить → выберите место

═══════════════════════════════════════════════════════════
⌨️ УПРАВЛЕНИЕ (v2.0)
═══════════════════════════════════════════════════════════

Навигация:
  🖱️ Колесо мыши → Плавное приближение (НОВОЕ!)
  🖱️ ЛКМ + перетаскивание → Перемещение карты
  🖱️ Наведение на город → Показать название (НОВОЕ!)

Создание города (НОВОЕ!):
  1. Кнопка "🏙️ Добавить город"
  2. Клик на карте в нужном месте
  3. Введите название города
  4. Enter или "✓ Подтвердить"

Создание маршрута:
  1. ПКМ по городу (начало)
  2. ЛКМ по другим городам (точки)
  3. Enter (завершить)
  4. Введите название маршрута

Работа с товарами (УПРОЩЕНО!):
  • Только название и наценка %
  • Убраны базовая и итоговая цены

Горячие клавиши:
  ESC → Отмена / Закрыть панель
  Enter → Подтвердить действие
