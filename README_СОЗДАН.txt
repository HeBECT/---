🚀 ПУБЛИКАЦИЯ НА GITHUB
═══════════════════════════════════════════════════════════

Шаги:

1. Инициализируйте Git:
   git init
   git add .
   git commit -m "Initial commit: Kenshi Trade Map v3.0"

2. Создайте репозиторий на GitHub:
   • Перейдите на github.com
   • Нажмите "New repository"
   • Название: kenshi-trade-map
   • Описание: Interactive trade route map for Kenshi
   • Public
   • Не добавляйте README (уже есть)

3. Свяжите с удаленным репозиторием:
   git remote add origin https://github.com/yourusername/kenshi-trade-map.git
   git branch -M main
   git push -u origin main

4. Настройте GitHub:
   • Добавьте описание
   • Добавьте темы (tags): kenshi, rust, trading, map, game
   • Включите Issues
   • Включите Discussions

5. Создайте релиз:
   • Перейдите в Releases
   • "Create a new release"
   • Tag: v3.0.0
   • Title: Kenshi Trade Map v3.0
   • Описание: см. историю версий
   • Прикрепите скомпилированные файлы
