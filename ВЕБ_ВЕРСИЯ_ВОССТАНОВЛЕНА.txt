🎮 БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════

Создать первый маршрут:

1. Откройте index.html
2. Нажмите "➕ Создать маршрут"
3. Кликните по первому городу (синяя точка)
4. Кликните по второму городу
5. Нажмите клавишу ENTER
6. Введите название маршрута
7. Выберите товар (опционально)
8. Укажите наценку (опционально)
9. Нажмите "✓ Создать"

✅ Ваш первый маршрут готов!

🎯 УПРАВЛЕНИЕ
═══════════════════════════════════════════════════════════

🖱️ Мышь:
   • ЛКМ + перетаскивание = перемещение карты
   • Колесо мыши = масштабирование
   • Клик по городу = выбор города
   • Наведение на маршрут = информация

⌨️ Клавиатура:
   • ENTER = завершить создание маршрута
   • ESC = отменить создание маршрута
   • F5 = обновить страницу
   • F11 = полноэкранный режим

🆚 ВЕБ-ВЕРСИЯ vs RUST-ВЕРСИЯ
═══════════════════════════════════════════════════════════

Веб-версия (HTML/CSS/JS):
  ✅ Проще запустить (просто открыть файл)
  ✅ Работает везде (любой браузер)
  ✅ Меньше размер (~50 КБ)
  ✅ Легче модифицировать
  ⚠️ Чуть медленнее

Rust-версия (Desktop):
  ✅ Быстрее работает (нативная скорость)
  ✅ Больше возможностей
  ✅ Автономное приложение
  ⚠️ Требует компиляции
  ⚠️ Больше размер (~5-10 МБ)

Подробное сравнение: СРАВНЕНИЕ_ВЕБ_И_RUST.md

🐛 УСТРАНЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════

Проблема: Белый экран
  → Откройте консоль (F12) и проверьте ошибки
  → Убедитесь, что все 3 файла в одной папке
  → Попробуйте другой браузер

Проблема: Не реагирует на мышь
  → Обновите страницу (Ctrl + F5)
  → Проверьте, что JavaScript включен

Проблема: Не сохраняются маршруты
  → Проверьте права на запись в папку
  → Попробуйте запустить через локальный сервер

🌐 АЛЬТЕРНАТИВНЫЕ СПОСОБЫ ЗАПУСКА
═══════════════════════════════════════════════════════════

Через Python:
  cd kenshi-trade-map-web
  python -m http.server 8000
  → Откройте: http://localhost:8000

Через Node.js:
  cd kenshi-trade-map-web
  npx http-server
  → Откройте: http://localhost:8080

Через PHP:
  cd kenshi-trade-map-web
  php -S localhost:8000
  → Откройте: http://localhost:8000

Через VS Code:
  1. Установите расширение "Live Server"
  2. Кликните правой кнопкой на index.html
  3. Выберите "Open with Live Server"
