# 🏴‍☠️ Kenshi Trade Map - Полный проект

Интерактивное приложение для создания и управления торговыми маршрутами в игре Kenshi.

## 📦 Состав проекта

Этот проект содержит **две версии** приложения:

### 1. 🌐 Web версия (HTML/CSS/JavaScript)
- **Файлы:** Были созданы ранее в корне проекта
- **Главный файл:** `index.html` (29 KB)
- **Документация:** README.md, ИНСТРУКЦИЯ.txt, ШПАРГАЛКА.txt и др.

### 2. 💻 Desktop версия (Rust) ⭐ НОВАЯ
- **Папка:** `kenshi-trade-map-rust/`
- **Язык:** Rust 2021 Edition
- **GUI:** egui + eframe

## 🚀 Rust версия - Быстрый старт

### Установка и запуск:

```powershell
# 1. Установите Rust
```

## 🎮 Использование

### Создание маршрута:
1. Загрузите карту Kenshi (PNG, JPG)
2. ПКМ на городе → "Создать маршрут"
3. ЛКМ по городам для добавления точек
4. Enter для завершения
5. Введите название маршрута

### Добавление товаров:
1. ЛКМ на линию маршрута
2. Откроется боковая панель
3. "+ Добавить товар"
4. Заполните: Товар, База, Наценка%
5. Итог рассчитается автоматически

---

## ⌨️ Управление

**Мышь:**
- Колесико - Зум
- ЛКМ + перетаскивание - Двигать карту
- ПКМ на городе - Контекстное меню
- ЛКМ на линию - Панель товаров

**Клавиатура:**
- Enter - Завершить маршрут
- Esc - Отменить / Закрыть панель

## 📊 Формула расчета

```
Итоговая цена = Базовая цена + (Базовая цена × Наценка% ÷ 100)
```

**Пример:**
- База: 100 кэтов
- Наценка: 50%
- Итог: 150 кэтов

## 🐛 Решение проблем

### Rust версия:

**"cargo" не найден:**
- Установите Rust с https://www.rust-lang.org/tools/install
- Перезапустите PowerShell
- Проверьте: `cargo --version`

**Долгая компиляция:**
- Это нормально! Первая сборка: 5-15 минут
- Последующие сборки: 30 секунд - 2 минуты

**Медленная работа:**
- Используйте: `cargo run --release`
- НЕ используйте: `cargo run` (без --release)

**Ошибка компиляции:**
```powershell
cargo clean
cargo build --release
```
