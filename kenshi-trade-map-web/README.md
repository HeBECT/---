# 🗺️ Kenshi Trade Map - Веб-версия

Интерактивная карта торговых маршрутов для игры Kenshi с возможностью создания и управления торговыми путями.

## ✨ Возможности

- 🗺️ **Интерактивная карта** с масштабированием и перемещением
- 🏙️ **22 города** из игры Kenshi
- 🛣️ **Создание торговых маршрутов** между городами
- 📦 **Управление товарами** с расчетом наценки
- 💾 **Сохранение и загрузка** маршрутов в JSON
- 🎨 **Красивый интерфейс** с градиентами и анимациями
- 📱 **Адаптивный дизайн** для разных устройств

## 🚀 Быстрый старт

### Вариант 1: Открыть напрямую

1. Откройте файл `index.html` в любом современном браузере
2. Готово! Приложение работает без сервера

### Вариант 2: Локальный сервер (рекомендуется)

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен npx)
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте: `http://localhost:8000`

## 🎮 Управление

### Навигация по карте
- **ЛКМ + перетаскивание** - перемещение карты
- **Колесо мыши** - масштабирование
- **Клик по городу** - выбор города

### Создание маршрута
1. Нажмите кнопку **"➕ Создать маршрут"**
2. Кликайте по городам или любым точкам на карте
3. Нажмите **Enter** для завершения маршрута
4. Заполните информацию о маршруте:
   - Название маршрута
   - Товар для торговли
   - Наценка (%)
5. Нажмите **"✓ Создать"**

### Просмотр маршрута
- Наведите курсор на линию маршрута
- Появится всплывающая подсказка с информацией
- Кнопка **"🗑️ Удалить"** для удаления маршрута

### Сохранение данных
- **💾 Сохранить** - экспорт маршрутов в JSON файл
- **📂 Загрузить** - импорт маршрутов из JSON файла

- Нет расчета расстояний и времени пути
