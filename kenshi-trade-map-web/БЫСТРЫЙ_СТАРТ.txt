🎯 ЗА 3 ШАГА К РАБОТАЮЩЕМУ ПРИЛОЖЕНИЮ:

┌─────────────────────────────────────────────────────────┐
│  ШАГ 1: ОТКРЫТЬ ФАЙЛ                                    │
└─────────────────────────────────────────────────────────┘

  1. Найдите файл "index.html" в папке проекта
  2. Дважды кликните по нему
  3. Файл откроется в браузере
  
  ✅ Готово! Приложение работает!

┌─────────────────────────────────────────────────────────┐
│  ШАГ 2: СОЗДАТЬ ПЕРВЫЙ МАРШРУТ                          │
└─────────────────────────────────────────────────────────┘

  1. Нажмите кнопку "➕ Создать маршрут"
  2. Кликните по любому городу (синяя точка)
  3. Кликните по другому городу
  4. Нажмите клавишу ENTER
  5. Введите название маршрута
  6. Нажмите "✓ Создать"
  
  ✅ Ваш первый маршрут создан!

┌─────────────────────────────────────────────────────────┐
│  ШАГ 3: СОХРАНИТЬ МАРШРУТЫ                              │
└─────────────────────────────────────────────────────────┘

  1. Нажмите кнопку "💾 Сохранить"
  2. Выберите место для сохранения файла
  3. Файл сохранится в формате JSON
  
  ✅ Ваши маршруты сохранены!

🎯 ПРИМЕР ИСПОЛЬЗОВАНИЯ:

  Задача: Создать торговый маршрут от Хаба до Стоата

  1. Откройте index.html
  2. Найдите город "Хаб" на карте (синяя точка)
  3. Нажмите "➕ Создать маршрут"
  4. Кликните по городу "Хаб"
  5. Найдите город "Стоат"
  6. Кликните по городу "Стоат"
  7. Нажмите ENTER
  8. Введите: "Хаб → Стоат"
  9. Выберите товар: "Железная руда"
  10. Укажите наценку: 50
  11. Нажмите "✓ Создать"
  
  ✅ Маршрут создан и отображается на карте!
