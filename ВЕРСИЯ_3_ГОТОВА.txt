🎮 БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════

1. Загрузите карту:
   📁 Загрузить карту → выберите изображение

2. Создайте города:
   🏙️ Добавить город → кликните на карте → введите название

3. Создайте маршрут:
   ПКМ по городу (начало) → ЛКМ по другому городу (конец)
   → Выберите цвет → Введите название → ✓ Создать

4. Создайте еще маршруты:
   Повторите шаг 3 для каждого маршрута
   От одного города можно создать много маршрутов!

5. Добавьте товары:
   Клик по линии маршрута → + Добавить товар

6. Сохраните:
   💾 Сохранить

═══════════════════════════════════════════════════════════
⌨️ УПРАВЛЕНИЕ
═══════════════════════════════════════════════════════════

Создание маршрута:
  1. ПКМ по городу (начало)
  2. ЛКМ по другому городу (конец)
  3. Выберите цвет (color picker или предустановленный)
  4. Введите название (или оставьте автоматическое)
  5. "✓ Создать"

Выбор цвета:
  • Кликните на color picker для любого цвета
  • Или выберите предустановленный:
    🟢 Зеленый  🔵 Синий  🟠 Оранжевый  🔴 Розовый
    🔷 Голубой  🟡 Желтый  🟣 Фиолетовый  🔺 Красный

Просмотр маршрута:
  • ЛКМ по линии маршрута
  • Откроется панель справа

Удаление маршрута:
  • Откройте панель маршрута
  • "🗑️ Удалить маршрут"

Навигация:
  • Колесо мыши - плавное приближение
  • ЛКМ + перетаскивание - перемещение
  • Наведение на город - показать название

Горячие клавиши:
  ESC - Отмена / Закрыть панель

Пример 1: Торговая сеть из Хаба
───────────────────────────────────────────────────────────
Создайте несколько маршрутов от одного города:

Хаб → Стоат (🟢 зеленый, железо +50%)
Хаб → Скваин (🔵 синий, ткань +30%)
Хаб → Блистер Хилл (🟠 оранжевый, еда +20%)
Хаб → Монгрел (🔴 красный, робототехника +100%)

Пример 2: Цветовая организация
───────────────────────────────────────────────────────────
Используйте цвета для группировки:

🟢 Зеленые = прибыльные маршруты
🔴 Красные = убыточные маршруты
🔵 Синие = нейтральные маршруты
🟡 Желтые = экспериментальные маршруты

Пример 3: Двусторонняя торговля
───────────────────────────────────────────────────────────
Создайте маршруты в обе стороны:

Хаб → Стоат (🟢 зеленый, железо туда)
Стоат → Хаб (🔵 синий, ткань обратно)
