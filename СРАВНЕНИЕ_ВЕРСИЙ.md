# Kenshi Trade Map - Сравнение версий

В этом проекте представлены **две версии** приложения для создания торговых маршрутов в игре Kenshi:

## 📦 Доступные версии

### 1. 🌐 Web версия (HTML/CSS/JavaScript)
**Папка:** `index.html` и документация в корне

**Преимущества:**
- ✅ Мгновенный запуск (просто откройте HTML файл)
- ✅ Не требует установки
- ✅ Работает в любом браузере
- ✅ Легкий вес (~29 KB)
- ✅ Простота распространения

**Недостатки:**
- ⚠️ Данные не сохраняются при закрытии
- ⚠️ Зависит от браузера
- ⚠️ Меньшая производительность

**Для кого:**
- Пользователи, которым нужно быстрое решение
- Те, кто не хочет устанавливать дополнительное ПО
- Для быстрого тестирования

---

### 2. 💻 Desktop версия (Rust)
**Папка:** `kenshi-trade-map-rust/`

**Преимущества:**
- ✅ Нативная производительность (очень быстро!)
- ✅ Сохранение/загрузка маршрутов в JSON
- ✅ Кроссплатформенность (Windows, Linux, macOS)
- ✅ Автономное приложение
- ✅ Плавная анимация 60 FPS
- ✅ Профессиональный интерфейс

**Недостатки:**
- ⚠️ Требует установки Rust для сборки
- ⚠️ Первая сборка занимает 5-15 минут
- ⚠️ Больший размер (~10-15 MB)

**Для кого:**
- Пользователи, которым нужна максимальная производительность
- Те, кто хочет сохранять данные
- Разработчики и энтузиасты Rust

## 🎯 Какую версию выбрать?

### Выберите **Web версию**, если:
- ✅ Вам нужно быстрое решение прямо сейчас
- ✅ Не хотите устанавливать дополнительное ПО
- ✅ Планируете использовать редко
- ✅ Не нужно сохранять данные между сессиями
- ✅ Хотите поделиться с друзьями (просто отправить HTML)

### Выберите **Rust версию**, если:
- ✅ Нужна максимальная производительность
- ✅ Хотите сохранять маршруты в файлы
- ✅ Планируете активно использовать приложение
- ✅ Интересуетесь программированием на Rust
- ✅ Нужно профессиональное десктопное приложение

## 🚀 Быстрый старт

### Web версия:
```
1. Откройте index.html в браузере
2. Загрузите карту
3. Начните создавать маршруты!
```

### Rust версия:
```powershell
1. Установите Rust: https://www.rust-lang.org/tools/install
2. cd kenshi-trade-map-rust
3. .\build.ps1
4. .\run.ps1
```
