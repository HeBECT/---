# Kenshi Trade Map - Desktop Application (Rust)

Кроссплатформенное десктопное приложение для создания и управления торговыми маршрутами в игре Kenshi.

## 🚀 Установка Rust

### Windows

1. Скачайте и установите Rust с официального сайта:
   ```
   https://www.rust-lang.org/tools/install
   ```

2. Запустите `rustup-init.exe` и следуйте инструкциям

3. Перезапустите терминал после установки

4. Проверьте установку:
   ```powershell
   cargo --version
   ```

### Linux / macOS

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
cargo --version
```

## 🔨 Сборка проекта

1. Перейдите в директорию проекта:
   ```powershell
   cd kenshi-trade-map-rust
   ```

2. Соберите проект:
   ```powershell
   cargo build --release
   ```

3. Запустите приложение:
   ```powershell
   cargo run --release
   ```

Исполняемый файл будет находиться в `target/release/kenshi-trade-map.exe`

## 📖 Использование

### Загрузка карты
1. Нажмите кнопку "📁 Загрузить карту"
2. Выберите изображение карты Kenshi (PNG, JPG)

### Навигация
- **Зум**: Колесико мыши
- **Перемещение**: Зажмите ЛКМ и двигайте мышью (не на городе)

### Создание маршрута
1. **ПКМ** на городе (сразу начнется создание маршрута)
2. **ЛКМ** по городам для добавления точек
3. **Enter** для завершения
4. Введите название маршрута
5. Нажмите "✓ Подтвердить"

### Управление товарами
1. **ЛКМ** на линию маршрута
2. Откроется боковая панель
3. Нажмите "+ Добавить товар"
4. Заполните поля:
   - Товар: название
   - База: базовая цена
   - Наценка %: процент наценки
   - Итог: рассчитывается автоматически

### Сохранение/Загрузка
- **💾 Сохранить маршруты**: Экспорт в JSON
- **📂 Загрузить маршруты**: Импорт из JSON

## ⌨️ Горячие клавиши

- `Enter` - Завершить создание маршрута
- `Esc` - Отменить действие / Закрыть панель
- `Колесико мыши` - Зум
- `ЛКМ + перетаскивание` - Перемещение карты
- `ПКМ на городе` - Контекстное меню

## 🔧 Технологии

- **Язык**: Rust 2021 Edition
- **GUI**: egui + eframe
- **Изображения**: image crate
- **Сериализация**: serde + serde_json
- **Диалоги**: rfd (native file dialogs)

## 📦 Зависимости

```toml
eframe = "0.27"      # GUI framework
egui = "0.27"        # Immediate mode GUI
image = "0.25"       # Image loading
serde = "1.0"        # Serialization
serde_json = "1.0"   # JSON support
rfd = "0.14"         # File dialogs
```

## 🚀 Оптимизация

Релизная сборка оптимизирована для максимальной производительности:
- LTO (Link Time Optimization)
- Оптимизация размера
- Strip symbols
- Codegen units = 1

## 📊 Формула расчета

```
Итоговая цена = Базовая цена + (Базовая цена × Наценка% ÷ 100)
```

## 🐛 Решение проблем

### Ошибка компиляции
```powershell
cargo clean
cargo build --release
```

### Не открывается окно
- Убедитесь, что установлены драйверы видеокарты
- Попробуйте запустить от имени администратора

### Медленная работа
- Используйте релизную сборку: `cargo run --release`
- Не используйте debug сборку для работы
