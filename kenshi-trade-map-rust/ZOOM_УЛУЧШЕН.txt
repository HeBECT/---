✅ 1. НОРМАЛИЗАЦИЯ СКОРОСТИ СКРОЛЛА
───────────────────────────────────────────────────────────

Было:
  • Скорость зависела от скорости прокрутки колеса
  • Быстрая прокрутка = резкий зум
  • Неконтролируемое поведение

Стало:
  • Скорость скролла ограничена: clamp(-50.0, 50.0)
  • Плавное и предсказуемое поведение
  • Одинаковая скорость независимо от прокрутки

Код:
```rust
let normalized_delta = scroll_delta.clamp(-50.0, 50.0);
```

✅ 2. ЗАМЕДЛЕНИЕ ZOOM FACTOR
───────────────────────────────────────────────────────────

Было:
  • zoom_factor = 1.15 (приближение)
  • zoom_factor = 1/1.15 = 0.87 (отдаление)
  • Слишком быстрое изменение масштаба

Стало:
  • zoom_factor = 1.08 (максимум)
  • zoom_factor = 0.92 (минимум)
  • Более плавное и контролируемое приближение

Формула:
```rust
let zoom_speed = 1.0 + (normalized_delta / 500.0);
let zoom_factor = zoom_speed.clamp(0.92, 1.08);
```

Изменение:
  • Было: ±15% за шаг
  • Стало: ±8% за шаг
  • Почти в 2 раза медленнее!

✅ 3. БОЛЕЕ ПЛАВНАЯ ИНТЕРПОЛЯЦИЯ
───────────────────────────────────────────────────────────

Было:
  • lerp_factor = 0.15 (15% за кадр)
  • Быстрая интерполяция

Стало:
  • lerp_factor = 0.12 (12% за кадр)
  • Более плавное движение
  • Меньше рывков

Код:
```rust
fn update_smooth_zoom(&mut self) {
    let lerp_factor = 0.12;  // Было 0.15
    self.zoom += (self.target_zoom - self.zoom) * lerp_factor;
}
```

✅ 4. УЛУЧШЕННАЯ КОРРЕКТИРОВКА ПОЗИЦИИ
───────────────────────────────────────────────────────────

Было:
  • offset_adjustment * 0.15

Стало:
  • offset_adjustment * 0.12
  • Более плавное смещение при зуме
  • Меньше "прыжков" камеры

🎮 КАК ЭТО РАБОТАЕТ
═══════════════════════════════════════════════════════════

Шаг 1: Получение скролла
  • Пользователь крутит колесо мыши
  • Получаем scroll_delta (может быть любым значением)

Шаг 2: Нормализация
  • Ограничиваем: clamp(-50.0, 50.0)
  • Теперь максимум ±50

Шаг 3: Расчет zoom_factor
  • zoom_speed = 1.0 + (normalized_delta / 500.0)
  • При +50: zoom_speed = 1.1
  • При -50: zoom_speed = 0.9
  • Ограничиваем: clamp(0.92, 1.08)

Шаг 4: Применение target_zoom
  • target_zoom *= zoom_factor
  • Ограничиваем: clamp(0.1, 3.0)

Шаг 5: Плавная интерполяция
  • Каждый кадр: zoom += (target_zoom - zoom) * 0.12
  • Плавное приближение к цели

Шаг 6: Корректировка позиции
  • Зум происходит относительно курсора
  • Плавная корректировка offset

🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ
═══════════════════════════════════════════════════════════

Формула нормализации:
```rust
let normalized_delta = scroll_delta.clamp(-50.0, 50.0);
```

Формула zoom_speed:
```rust
let zoom_speed = 1.0 + (normalized_delta / 500.0);
// При +50: 1.0 + 0.1 = 1.1
// При -50: 1.0 - 0.1 = 0.9
```

Ограничение zoom_factor:
```rust
let zoom_factor = zoom_speed.clamp(0.92, 1.08);
// Минимум: 0.92 (отдаление на 8%)
// Максимум: 1.08 (приближение на 8%)
```

Интерполяция:
```rust
self.zoom += (self.target_zoom - self.zoom) * 0.12;
// 12% от разницы каждый кадр
// При 60 FPS: полное приближение за ~0.5 секунды
```
