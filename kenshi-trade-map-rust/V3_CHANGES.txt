🎯 ОСНОВНЫЕ ИЗМЕНЕНИЯ v3.0

✅ 1. НЕОГРАНИЧЕННОЕ КОЛИЧЕСТВО МАРШРУТОВ ОТ ОДНОГО ГОРОДА
───────────────────────────────────────────────────────────

Было (v2.0):
  • Можно было создавать маршруты с множеством точек
  • Один маршрут мог проходить через много городов

Стало (v3.0):
  • Один маршрут = ТОЛЬКО 2 точки (начало и конец)
  • От одного города можно создать НЕОГРАНИЧЕННОЕ количество маршрутов
  • Каждый маршрут независим

Преимущества:
  ✅ Проще управлять маршрутами
  ✅ Четкое обозначение начала и конца
  ✅ Можно создать сеть маршрутов
  ✅ Легче добавлять товары

Пример:
  Хаб → Стоат (маршрут 1)
  Хаб → Скваин (маршрут 2)
  Хаб → Блистер Хилл (маршрут 3)
  И так далее...

✅ 2. ВИЗУАЛЬНОЕ ОБОЗНАЧЕНИЕ НАЧАЛА И КОНЦА
───────────────────────────────────────────────────────────

Новые визуальные элементы:
  • Стрелка на конце маршрута (показывает направление)
  • Точка в начале маршрута
  • Точка в конце маршрута
  • Четкое направление торговли

Теперь видно:
  ✅ Откуда начинается маршрут
  ✅ Куда идет маршрут
  ✅ Направление торговли

✅ 5. УПРОЩЕННАЯ СТРУКТУРА ДАННЫХ
───────────────────────────────────────────────────────────

Было (v2.0):
  • Маршрут содержал массив точек
  • Сегменты с товарами
  • Сложная структура

Стало (v3.0):
  • Маршрут содержит только 2 точки
  • Товары привязаны к маршруту напрямую
  • Простая структура

Структура TradeRoute:
```rust
struct TradeRoute {
    id: usize,
    name: String,
    start_point: Pos2,
    end_point: Pos2,
    start_city_name: String,
    end_city_name: String,
    color: [u8; 3],
    items: Vec<TradeItem>,
}
```

🎮 НОВЫЙ РАБОЧИЙ ПРОЦЕСС
═══════════════════════════════════════════════════════════

1. Создание маршрута:
   ПКМ по городу (начало) → ЛКМ по другому городу (конец)

2. Настройка маршрута:
   • Введите название (или оставьте автоматическое)
   • Выберите цвет (color picker или предустановленный)
   • Нажмите "✓ Создать"

3. Добавление товаров:
   • Кликните по линии маршрута
   • Откроется панель справа
   • "+ Добавить товар"
   • Введите название и наценку

4. Создание нескольких маршрутов:
   • Повторите шаги 1-3 для каждого маршрута
   • От одного города можно создать сколько угодно маршрутов

⌨️ УПРАВЛЕНИЕ

Создание маршрута:

  • "🗑️ Удалить маршрут"

Горячие клавиши:
  ESC - Отмена / Закрыть панель
