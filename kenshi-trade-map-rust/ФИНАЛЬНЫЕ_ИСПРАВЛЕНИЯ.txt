═══════════════════════════════════════════════════════════════════
                    📝 ТЕХНИЧЕСКИЕ ДЕТАЛИ
═══════════════════════════════════════════════════════════════════

1. СЕРИАЛИЗАЦИЯ Pos2:
   
   Было:
   #[serde(skip)]
   points: Vec<Pos2>,
   
   Стало:
   #[serde(skip, default)]
   points: Vec<Pos2>,
   
   Атрибут "default" необходим для десериализации!


2. ПЕРЕСОЗДАНИЕ POINTS:
   
   При загрузке маршрутов из JSON, points пересоздаются
   из city_names:
   
   for route in &mut routes {
       route.points.clear();
       for city_name in &route.city_names {
           if let Some(city) = self.cities.iter()
               .find(|c| &c.name == city_name) {
               route.points.push(Pos2::new(city.x, city.y));
           }
       }
   }


3. ПРЕДУПРЕЖДЕНИЯ:
   
   Неиспользуемые переменные помечены префиксом "_"
   чтобы компилятор не выдавал предупреждения.
