|----------------|------------|-------------|

## 🌐 Веб-версия (HTML/CSS/JavaScript)

### ✅ Преимущества

1. **Простота запуска**
   - Открыть файл `index.html` в браузере
   - Не требует установки
   - Работает сразу "из коробки"

2. **Кроссплатформенность**
   - Windows, macOS, Linux
   - Любой современный браузер
   - Даже на мобильных устройствах

3. **Легкость распространения**
   - Просто отправить файлы
   - Можно разместить на хостинге
   - Доступ через интернет

4. **Малый размер**
   - Всего 3 файла (~50 КБ)
   - Быстрая загрузка
   - Не занимает место на диске

5. **Простота модификации**
   - Открытый код
   - Легко редактировать
   - Не требует перекомпиляции

### ❌ Недостатки

1. **Производительность**
   - Медленнее нативного приложения
   - Ограничения JavaScript
   - Зависит от браузера

2. **Функциональность**
   - Ограничения безопасности браузера
   - Нет доступа к системным ресурсам
   - Ограниченная работа с файлами

3. **Зависимость от браузера**
   - Требует установленный браузер
   - Разное поведение в разных браузерах
   - Нужен JavaScript

### 📦 Что входит в веб-версию

```
kenshi-trade-map-web/
├── index.html           # Главная страница (15 КБ)
├── styles.css           # Стили (8 КБ)
├── app.js               # Логика (25 КБ)
├── README.md            # Документация
├── УСТАНОВКА.txt        # Руководство по установке
├── РУКОВОДСТВО.txt      # Руководство пользователя
└── БЫСТРЫЙ_СТАРТ.txt    # Быстрый старт
```

### 🚀 Как запустить

```bash
# Вариант 1: Просто открыть
Дважды кликнуть по index.html

# Вариант 2: Локальный сервер (Python)
python -m http.server 8000

# Вариант 3: Локальный сервер (Node.js)
npx http-server
```

## 🦀 Rust-версия (Desktop)

### ✅ Преимущества

1. **Высокая производительность**
   - Нативная скорость выполнения
   - Эффективное использование памяти
   - Плавная работа с большими данными

2. **Расширенные возможности**
   - Полный доступ к файловой системе
   - Системные уведомления
   - Горячие клавиши
   - Работа в фоне

3. **Автономность**
   - Не требует браузера
   - Работает офлайн
   - Независимое приложение

4. **Профессиональный вид**
   - Нативный интерфейс
   - Интеграция с ОС
   - Иконка в трее

5. **Безопасность**
   - Строгая типизация Rust
   - Защита от ошибок памяти
   - Надежность

### ❌ Недостатки

1. **Сложность установки**
   - Требует установки Rust
   - Нужна компиляция
   - Долгая первая сборка (~5-10 минут)

2. **Размер**
   - Исполняемый файл ~5-10 МБ
   - Зависимости занимают место
   - Долгая компиляция

3. **Распространение**
   - Нужно компилировать для каждой ОС
   - Сложнее обновлять
   - Требует технических знаний

4. **Модификация**
   - Нужно знать Rust
   - Требуется перекомпиляция
   - Сложнее для новичков

### 📦 Что входит в Rust-версию

```
kenshi-trade-map-rust/
├── src/
│   └── main.rs          # Исходный код (~600 строк)
├── Cargo.toml           # Конфигурация проекта
├── Cargo.lock           # Зависимости
├── build.ps1            # Скрипт сборки
├── run.ps1              # Скрипт запуска
├── README.md            # Документация
└── target/              # Скомпилированные файлы
    └── release/
        └── kenshi-trade-map.exe  # Исполняемый файл
```

### 🚀 Как запустить

```bash
# Установка Rust (один раз)
# Скачать с https://rustup.rs/

# Компиляция и запуск
cd kenshi-trade-map-rust
cargo build --release
cargo run --release

# Или использовать скрипты
./build.ps1
./run.ps1
```

## 💻 Технические характеристики

### Веб-версия

```
Технологии:
  - HTML5 Canvas
  - Vanilla JavaScript (ES6+)
  - CSS3 (Grid, Flexbox, Animations)

Размер файлов:
  - index.html: ~15 КБ
  - styles.css: ~8 КБ
  - app.js: ~25 КБ
  - Итого: ~50 КБ

Требования:
  - Современный браузер (Chrome 90+, Firefox 88+)
  - JavaScript включен
  - Разрешение: 1024x768+

Производительность:
  - 60 FPS при <50 маршрутах
  - Плавное масштабирование
  - Быстрая отрисовка
```

### Rust-версия

```
Технологии:
  - Rust 1.70+
  - egui (GUI фреймворк)
  - eframe (приложение)
  - serde (сериализация)

Размер файлов:
  - Исходный код: ~600 строк
  - Исполняемый файл: ~5-10 МБ
  - Зависимости: ~50 МБ (при сборке)

Требования:
  - Windows 10+ / Linux / macOS
  - 4 ГБ RAM
  - Rust toolchain (для сборки)

Производительность:
  - 60 FPS при любом количестве маршрутов
  - Мгновенное масштабирование
  - Нативная скорость
```

## 🔄 Миграция между версиями

### Из веб-версии в Rust-версию

✅ **Совместимость данных**: Обе версии используют одинаковый формат JSON  
✅ **Простой перенос**: Сохраните маршруты в веб-версии, загрузите в Rust-версии  
✅ **Без потерь**: Все данные сохраняются  

### Из Rust-версии в веб-версию

✅ **Совместимость данных**: Формат JSON идентичен  
✅ **Простой перенос**: Сохраните в Rust-версии, загрузите в веб-версии  
✅ **Без потерь**: Все данные сохраняются
